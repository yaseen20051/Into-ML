# ML Model Development & Deployment

This workspace contains machine learning projects focused on developing, training, and deploying models as production-ready APIs.

Each project follows a complete end-to-end pipeline:
- **Data exploration & preprocessing** - cleaning, feature engineering, EDA
- **Model training & evaluation** - classical ML algorithms
- **Model serialization** - saving trained models with pickle
- **API deployment** - serving predictions via Flask REST APIs
